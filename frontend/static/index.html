RSS Filter

Tired of not being able to catch up with all your RSS feeds? This tool will help you filter out the noise...

rssfilter is a recommendation system based on user read articles. By replacing the original RSS feed URLs, it starts reducing the number of unwanted items and tracking user read articles. It uses LLM embeddings and machine learning to recommend similar articles.

You can automatically convert all your existing feeds by uploading an OPML export from your RSS client.
